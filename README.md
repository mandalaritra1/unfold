Use latest version of the tool in `unfolder_v4.ipynb`. New tools are stored in `unfold_tool/`

For the rho workflow:

- Run `notebooks/unfolder_v4_rho.ipynb` only to produce outputs under `outputs/rho/`.
- Review saved plots in `notebooks/rho_review.ipynb`.
- Build a static scrollable gallery with `python3 outputs/build_rho_gallery.py`.
- Open `outputs/rho/index.html` in a browser when you want a fast overview of many plots.
